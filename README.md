# case-dadosfera
Este projeto visa completar os pontos estabelecidos no Case Técnico da empresa Dadosfera. Para isso, é utilizada uma fonte de dados desestruturada que passa por certos processso de ETL e uso de LLM para enriquecimento semântico. Ao fim, uma forma estruturada pode ser utilizada para EDA, ou seja, análise e visualizações.
